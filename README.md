## Ejercicios desarrollados

### Ejercicio 1: Navegación y creación de directorios
Se usaron comandos como `pwd`, `cd`, `mkdir -p` para identificar rutas, desplazarse por el sistema de archivos y crear carpetas anidadas.

### Ejercicio 2: Creación, redirección y lectura de archivos
Se usaron `echo`, `cat`, `less`, `mv`, `ls` y `ls -a` para crear archivos, unir contenido, visualizar información y convertir un archivo en oculto.

### Ejercicio 3: Privilegios administrativos
Se comparó el acceso con `cat /etc/shadow` y `sudo cat /etc/shadow`, evidenciando la diferencia entre permisos de usuario normal y permisos administrativos.

### Ejercicio 4: Automatización mediante scripts
Se creó el script `info_entorno.sh`, se otorgaron permisos de ejecución con `chmod +x` y se ejecutó con `./info_entorno.sh`.

### Ejercicio 5: Práctica autodidacta
Se ejecutaron 10 comandos adicionales: `touch`, `cp`, `head`, `tail`, `wc`, `grep`, `find`, `date`, `whoami` y `uname -a`.
